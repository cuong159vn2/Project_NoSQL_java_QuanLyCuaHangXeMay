	HƯỚNG DẪN SỬ DỤNG

   Yêu cầu:
- Cài đặt netbeans IDE 8.2 RC

- Cài đặt mongoBD và mongoBD compass

- Cài đặt SQLServer 2012

- Cài đặt ireport và jasperReport bản mới nhất
     + File cài đặt ireport (plugins): 
	* link cài đặt:
		https://drive.google.com/file/d/1ZIMiYQyyKcCf_CLp4y1vdi0SPISGybIR/view?usp=sharing
	* Link hướng dẫn cài đặt: (phút 2:30)
		https://www.youtube.com/watch?v=xs_OG9yYqWI
     + jasperReport chỉ cần có thư viện hỗ trợ và đã có trong mục thuvien trong file bài nộp.
      
   Hướng dẫn chạy:
B1: Tạo 1 database trên mongoDB đặt tên là QLCHXM
B2: Import các file json trong mục database
B3: Chạy file QuanLyCuaHangXeMay.sql trên sqlserver 2012 trong mục database
B4: Mở Project QLCHXM và add các thư viện hỗ trợ vào library từ mục thuvien trong file bài nộp
B5: chạy class "Main" trong package "Main"
B6: Đăng nhập:
	nhân viên quản lý:
		mã nhân viên: NV01
		mật khẩu:     123
	nhân viên bán hàng:
		mã nhân viên: NV02
		mật khẩu:     123
 ------> chạy chương trình theo các chức năng có sẵn trên form.
